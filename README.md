# DevOPs_Projects
Projects for Cloud Operations Engineer, DevOps Engineer, AWS, Jenkins, Docker, Kubernetes, Terraform, CI/CD

git branch
	- show branch name you currently working on

git checkout feature/devs_projects
	- switch one branch to another branch

git status
	 - showing modified files names

git add *
	- Prepare to commit/add files to current feature branch

git commit -m "files comments"
	- added comments & then commit modified files

git push
	- push all the changes into current git branch (i.e. feature/devs_projects)
